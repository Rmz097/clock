🌦️ GUÍA INTELIGENTE DEL CLIMA

Aprende a interpretar el clima como si fueras meteorólogo: entiende cuándo hará calor incómodo, si conviene salir, hacer ejercicio, llevar paraguas o protegerte del sol.

🥵 ¿Hará calor?

Mira primero: Punto de rocío + sensación térmica

🌧️ ¿Lloverá?

Revisa: Probabilidad + nubosidad

🏃 ¿Buen clima?

Ideal: bajo UV + poco viento + rocío cómodo

☀️ ¿Sol peligroso?

Si el UV es mayor a 6: usa bloqueador

⚡ Método rápido para entender el clima

💦 Punto de rocío > 20°C Bochorno
☀️ Índice UV ≥ 6 Protección solar
🌧️ Lluvia ≥ 60% Lleva paraguas
🌬️ Viento ≥ 30 km/h Molesto
👀 Visibilidad < 5 km Precaución

💦 Punto de rocío (el más importante)

Indica qué tan húmedo se siente realmente el aire. Es el mejor indicador para saber si sentirás calor incómodo.

¿Por qué importa?

  • Explica el bochorno
  • Indica qué tanto sudarás
  • Afecta energía y sueño
  • Hace que el calor “se pegue”

Interpretación:

  • ≤ 10°C → Aire seco
  • 10–15°C → Muy agradable
  • 16–18°C → Algo húmedo
  • 19–21°C → Húmedo
  • 22–24°C → Muy húmedo
  • ≥ 25°C → Sofocante

💡 Si el punto de rocío supera 20°C, el cuerpo ya batalla para enfriarse.

💧 Humedad (%)

Cantidad de vapor de agua presente en el aire.

¿Qué provoca?

  • Sudor pegajoso
  • Cabello esponjado
  • Ambiente pesado
  • Mayor sensación de calor

Valores:

  • 0–30% → Muy seco
  • 30–60% → Cómodo
  • 60–80% → Húmedo
  • 80–100% → Muy húmedo

⚠️ La humedad sola puede engañar; el punto de rocío es más útil.

🌡️ Sensación térmica

Es la temperatura que realmente siente tu cuerpo.

¿Qué la modifica?

  • Humedad
  • Viento
  • Radiación solar

Cómo interpretarla:

  • Mayor que la real → humedad/calor
  • Menor que la real → viento/frío

🥵 Si la sensación es 3–5°C mayor, probablemente sentirás cansancio rápido.

🌬️ Viento

¿Para qué sirve?

  • Enfría el cuerpo
  • Ayuda a dispersar contaminación
  • Afecta ejercicio y ciclismo

Velocidades:

  • 0–10 km/h → Calmado
  • 10–20 km/h → Brisa ligera
  • 20–40 km/h → Moderado
  • 40–60 km/h → Fuerte
  • 60+ km/h → Peligroso

🌬️ Más viento = más fresco, pero también más resequedad.

🌧️ Probabilidad de lluvia

¿Cómo entenderla?

  • 0–20% → Muy poco probable
  • 30–50% → Podría llover
  • 60–80% → Probable
  • 90–100% → Casi seguro

Consejos:

  • ≥ 50% → considera paraguas
  • ≥ 70% → evita ropa clara o tenis delicados
  • Tormentas → evita actividades al aire libre

☀️ Índice UV

Mide qué tan intensa y peligrosa es la radiación solar.

Niveles:

  • 0–2 → Bajo
  • 3–5 → Moderado
  • 6–7 → Alto
  • 8–10 → Muy alto
  • 11+ → Extremo

Recomendaciones:

  • UV 6+ → bloqueador
  • UV 8+ → evita sol prolongado
  • UV 10+ → busca sombra

☀️ Aunque esté nublado, el UV puede seguir alto.

🔭 Visibilidad

Qué tan lejos puedes ver claramente.

  • 10+ km → Excelente
  • 5–10 km → Normal
  • 2–5 km → Reducida
  • < 2 km → Mala

Puede indicar:

  • Niebla
  • Contaminación
  • Polvo
  • Lluvia intensa

🕐 Ciclo del día

🌄 Amanecer

  • Hora más fresca
  • Ideal para correr
  • Menor radiación UV

🌇 Atardecer

  • Temperatura más agradable
  • Perfecto para caminar
  • Menos calor solar

😷 Calidad del aire

Indica qué tan limpio o contaminado está el aire y cómo puede afectar tu salud o actividades al aire libre.

Niveles:

  • Desconocido → No hay suficientes datos disponibles
  • Excelente → Aire muy limpio, ideal para cualquier actividad
  • Buena → Sin riesgo para la mayoría de personas
  • Moderada → Personas sensibles podrían presentar molestias leves
  • Mala → Reduce ejercicio intenso al aire libre
  • Muy mala → Evita actividades prolongadas en exteriores
  • Extremadamente mala → Permanece en interiores si es posible

😷 Mientras peor sea la calidad del aire, mayor fatiga, irritación o dificultad para respirar puedes sentir.

🧠 Cómo usar el clima en tu vida diaria

👕 Cómo vestirte

  • Rocío alto → ropa ligera
  • Viento fuerte → chamarra ligera
  • UV alto → gorra/lentes

🏃 Ejercicio

  • Mañana = mejor horario
  • UV bajo = más seguro
  • Rocío alto = te fatigas más

🌳 Actividades al aire libre

  • Viento moderado = agradable
  • Lluvia alta = mejor cancelar
  • Atardecer = clima ideal

🏠 En casa

  • Humedad alta → moho
  • Mucho calor → ventilar
  • Rocío alto → usar ventilador/AC
